## Installation

**Clone the repository:**
   ```bash
   git clone git@github.com:sinaghadrico/user-generator-test-assignment.git
   cd user-generator-test-assignment
 ```

 **Install packages:**
   ```bash
  yarn install
 ```
 **Run project:**
   ```bash
  yarn dev
 ```
  **Run tests:**
   ```bash
  yarn test
 ```
